Показаны примеры модуля Flask из Python.

1. Запустите файл app.py.
2. Перейдите по ссылке http://127.0.0.1:5000/api/weather.
3. Вы можете использовать разные маршруты такие как:\n
    /hello/name |  ВМЕСТО name ВВЕДИТЕ ВАШЕ ИМЯ.\n
    /city?city=city |  ВМЕСТО ТРЕТЬЕГО city ВВЕДИТЕ ВАШ ГОРОД.\n
    /now    |  С помощью этого маршрута вы сможете узнать время в настоящий момент и, случайно выбранную температуру
